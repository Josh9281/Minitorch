# Minitorch: A Minimal PyTorch Implementation

This project implements a deep learning framework from scratch, inspired by PyTorch.

- Docs: https://minitorch.github.io/

## Modules

- `module1`: Tensor operations and basic math ops
- `module2`: Automatic differentiation engine
- `module3`: Neural network layers and backpropagation
- `module4`: Training framework + MNIST classification

## Examples

Includes notebooks and test cases demonstrating:
- Training a 2-layer MLP on MNIST
- Computing gradients manually vs automatically
